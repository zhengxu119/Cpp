C++学习
=====
<p> 
C++优势：不仅提供一些高级的抽象机制，而且根基和C在同一层面；同时可以支持4种不同的编程风格:C风格、基于对象、面向对象和泛型.<br />
C++11出现前的缺陷:缺少自动内存管理和对象级别的消息发送机制.<br />
C++11出现后的优势:增强了语言机制和完善的标准库，为C++语言的编程风格带来了革命性的变化，写出的代码简洁优雅，并引入了各种类型智能指针.
</p>
